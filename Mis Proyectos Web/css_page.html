    En CSS verémos la parte del embellesimiento del frontend. Donde se podrá personalizar la página a gusto del editor; modificar su tamaño, color, posición, entre otros atributos que le darán una fachada que represente dicha página web.

    Selectores en CSS

    Universal: * como atributo. Para marcar todos los elementos.
    De Tipo: Para seleccionar los titulos, parrafos botton's ó cualquier otro objéto que se integre en el código HTML (los Tag's).
    Clases: (class como atributo y en CSS con un ".sample") Es para darle identidad al objeto y modificar este usando el nombre que se le haya dado.
    ID: (id como atributo y en CSS con "#") Este solo se debe usar para Identificar un objeto como regla.
    Por Atributo:Con este creamos un atributo en el HTML que usaremos en el código CSS (se representa con "[sample]") para modificarlo.
    Decendiente:En este se usarán los tag's por orden jerárquico. Para separar elementos dentro de otros elementos sin que esos alteren a otros tag's dentro o fuera de el tag que lo encierra.
    Pseudo-clases:Son modificaciones especiales que se agregan a cualquier elemento en el codigo, primeramente identificando como una clase. Las mas usadas son: visited, link, hover y/o active.
    Estilos en linea: se representa con "style".
    Importante: Se representa como "!important" y este supera en orden jerarquico a todos los demas selectores.

    Orden jerárquico de selectores:

    1. !important
    2. style
    3. id
    4. class
    5. pseudo-class
    6. atributes
    7. element
    8. pseudo-element

    Metodologia BEM:

    La metodología BEM divide la interfaz de usuario en bloques independientes para crear componentes escalables y reutilizables.

    BEM (Block, Element, Modifier o Bloque, Elemento, Modificador) es una metodología ágil de desarrollo basada en componentes. Fue creada por la empresa Yandex para desarrollar sitios en poco tiempo y con un soporte de largo plazo.

    El objetivo final de BEM es dividir la interfaz de usuario en bloques independientes para crear componentes escalables y reutilizables. Propone un estilo descriptivo para nombrar cada una de las clases a utilizar, permitiendo así crear un estructura de código consistente.

    REM y EM

    rem y em como atributo para las medidas en relatividad a la cantidad de pixeles que este ocupa, es decir: que 1em equivale a 16px por defecto, y esta se puede personalizar colocandole otro tamaño a la caja contenedora. Esto es usado para la adaptabilidad al tamaño de la ventana (Responsive Desing).

    Como tambien lo estan los Viewport (representados como "vw" en la hoja de CSS) para determinar el porcentaje que ocupe el objetpo en patalla.
