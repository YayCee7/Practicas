Conceptos de lenguaje HTML:

"DOCTYPE" (tipo de documento).
"HTML" (Hyper Text Multi Language).
"SEO" (Search Engine Optimization).

Tipos de escritura para HTML:

b: para negrita.
i: para cursiva.
strike: para tachada.
small: para chiquita.
a: para enlaces de re-direccion, de manera local con rutas de la carpeta contenedora y externa con la referencia de https.
